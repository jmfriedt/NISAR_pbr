## Calibration of the Ardusimple OEM multiband GNSS antenna

Using a LibreVNA, the active antenna is powered by a biasT connected
to the VNA input port 2, while port 1 radiates at -40 dBm a signal
in the L band to assess the power received by the active antenna. This
measurements allows for validating that the NISAR signal falls well
within the lower-L band transfer function of the antenna.
